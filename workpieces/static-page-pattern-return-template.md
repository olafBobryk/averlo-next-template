# Workpiece: Static Page Pattern Return Template

Status: planned

Role: template

Guide: ../_guides/artifacts/workpiece.md

## Intent

Template the return step where a static page worker records prep mode, review
route, changed files, verification evidence, review findings, and reusable
learning without prematurely turning page-specific facts into scaffold defaults.

## Acceptance

- Return evidence lists the entry recommendation, accepted prep-mode decision,
  changed files, route paths, source references, review routing decision, review
  evidence, and verification commands.
- Section-defined contract decisions are summarized without turning the page
  implementation into the only source of truth.
- Learning is split into path-level candidates, project-level lessons, parked
  scaffold research, and do-not-promote details.
- Missing source material, ownership conflicts, or cross-shape responsibilities
  are recorded as gaps.
- The return packet may recommend that the steward invoke the optional quality
  harness when scroll performance, layout stability, review friction, or
  repeated visual regressions justify a measured follow-up.
- No downstream page work is started by the return packet itself.

## Tests

- Check return evidence includes accepted prep-mode and review routing
  decisions.
- Check reusable lessons are split into clear tiers.
- Check unresolved gaps are explicit.
- Check verification and review evidence cover the chosen route.
- Check any quality-harness recommendation names the page risk it is meant to
  measure.
- Check no downstream page work is started by the return packet itself.

## Artifacts

- Future page worker return note.
- Proposed path-level updates.
- Project-level lessons.
- Parked scaffold research notes.
- Do-not-promote notes.
- Verification summary.

## Notes

- This is the place where page-route learning is classified before anything is
  promoted.
- The steward owns whether a quality-harness follow-up runs; the pattern return
  only recommends it when the page evidence justifies the extra loop.
